Sobre:

Projeto de Api para gerenciar senhas feita com Django, Postgresql e pykeepass

Permite salvar senhas em um arquivo criptografado com senha e gerar senhas aleatorias

Estrutura:

/ Pagina inicial ['GET']

/about Pagina sobre o projeto ['GET']

/login Pagina de login e cadastro ['GET', 'POST']

/pwd/ Pagina do gerenciador de senhas ['GET', 'POST']

/pwd/new Pagina para gerar senhas randomicas ['GET', 'POST']
